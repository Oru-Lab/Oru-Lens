## ğŸ“– Usage Guide

### 1. Access the Scanner
Navigate to the main page and click "New Lens" to start analyzing a contract.

### 2. Input Contract Details
- **Contract Address**: Paste any Ethereum-compatible contract address
- **Contract ABI**: Upload or paste the ABI JSON
- **Network**: Select the appropriate blockchain network

### 3. Explore Contract Functions
The dashboard automatically categorizes all contract functions:
- **Read Functions**: View-only operations (no gas cost)
- **Write Functions**: State-changing operations (require gas)
- **Events**: Contract events and listeners

### 4. Copy Integration Code
For each function, switch between framework tabs:
- **wagmi**: Modern React hooks for Ethereum
- **ethers.js**: Traditional Web3 library
- **Viem**: Lightweight Ethereum client

Click "Copy" to get production-ready code snippets.
